# 第 18 章：JVM 原理

## 学习目标

- 理解 JVM 内存结构与类加载过程
- 了解垃圾回收与常见调优思路

## 核心知识点

### JVM 内存结构（运行时数据区）

| 区域            | 说明                                   |
| --------------- | -------------------------------------- |
| 堆（Heap）      | 对象实例，GC 主要区域                  |
| 方法区 / 元空间 | 类元信息、常量、静态变量               |
| 虚拟机栈        | 方法调用，栈帧（局部变量表、操作数栈） |
| 本地方法栈      | Native 方法                            |
| 程序计数器      | 当前线程字节码行号                     |

### 对象创建与内存布局

- `new` → 类加载检查 → 分配内存 → 初始化 → 设置对象头
- 对象：对象头 + 实例数据 + 对齐填充

### 类加载机制

```
加载 → 验证 → 准备 → 解析 → 初始化 → 使用 → 卸载
```

- **双亲委派**：Bootstrap → Extension → Application → 自定义
- 打破双亲委派：SPI、OSGi、热部署

### 垃圾回收

- **判断存活**：引用计数（Java 不用）、可达性分析（GC Roots）
- **回收算法**：标记-清除、标记-整理、复制算法
- **分代收集**：新生代（Eden/S0/S1）、老年代

### 常见垃圾收集器

| 收集器           | 特点                 |
| ---------------- | -------------------- |
| Serial           | 单线程，STW          |
| Parallel         | 多线程吞吐优先       |
| CMS              | 并发标记清除，低延迟 |
| G1               | 分区，可预测停顿     |
| ZGC / Shenandoah | 超低延迟（Java 11+） |

### JVM 调优入门

- 堆参数：`-Xms`、`-Xmx`、`-Xmn`
- GC 日志：`-Xlog:gc*`
- 工具：`jps`、`jstat`、`jmap`、`jstack`、VisualVM、Arthas

### 常见问题

- **OOM**：堆溢出、元空间、直接内存、栈溢出
- **内存泄漏**：对象无法被 GC 回收（集合持有引用、未关闭资源）

## 重点难点

- 栈存局部变量与引用，堆存对象本身
- Full GC 停顿对延迟敏感系统的影响
