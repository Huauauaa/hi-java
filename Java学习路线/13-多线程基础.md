# 第 13 章：多线程基础

## 学习目标

- 理解进程与线程、并发与并行
- 掌握线程创建与基本同步

## 核心知识点

### 基本概念

- **进程**：资源分配单位
- **线程**：CPU 调度单位，共享进程内存
- 并发：单核交替执行；并行：多核同时执行

### 创建线程

```java
// 1. 继承 Thread
class MyThread extends Thread {
    public void run() { /* ... */ }
}

// 2. 实现 Runnable（推荐）
new Thread(() -> System.out.println("run")).start();

// 3. 实现 Callable + FutureTask（有返回值）
FutureTask<Integer> task = new FutureTask<>(() -> 42);
new Thread(task).start();
int result = task.get();
```

### 线程生命周期

```
NEW → RUNNABLE ⇄ BLOCKED/WAITING/TIMED_WAITING → TERMINATED
```

### 常用方法

- `start()` vs `run()`：start 启动新线程
- `sleep(ms)`：不释放锁
- `yield()`：让出 CPU
- `join()`：等待线程结束
- `interrupt()`：中断标志

### synchronized

```java
// 同步方法
public synchronized void method() { /* ... */ }

// 同步块
synchronized (lockObj) {
    // 临界区
}
```

- 保证互斥与可见性（配合内存模型）
- 锁对象：实例方法锁 this，静态方法锁 Class

### 线程安全

- 竞态条件：多线程读写共享可变状态
- 不可变对象、线程封闭、同步机制

### ThreadLocal

- 线程私有变量副本
- 典型：数据库连接、用户上下文

## 重点难点

- `sleep` 不释放锁，`wait` 释放锁（须在 synchronized 内）
- 不要直接调用 `Thread.stop()`（已废弃）
