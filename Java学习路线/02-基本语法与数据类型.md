# 第 02 章：基本语法与数据类型

## 学习目标

- 掌握 Java 基本语法规则
- 区分基本类型与引用类型
- 理解变量、常量与作用域

## 核心知识点

### 标识符与命名规范

- 字母、数字、`_`、`$`；不能以数字开头
- 驼峰命名：类名大驼峰，方法/变量小驼峰，常量全大写下划线

### 八种基本数据类型

| 类型 | 字节 | 范围/说明 |
|------|------|-----------|
| `byte` | 1 | -128 ~ 127 |
| `short` | 2 | ±32767 |
| `int` | 4 | 最常用整数 |
| `long` | 8 | 后缀 `L` |
| `float` | 4 | 后缀 `f` |
| `double` | 8 | 默认浮点 |
| `char` | 2 | Unicode 字符 |
| `boolean` | — | `true` / `false` |

### 引用类型

- 类、接口、数组、枚举
- 默认值为 `null`；通过 `new` 创建对象

### 类型转换

- **自动提升**：小范围 → 大范围（`byte` → `int` → `long` → `float` → `double`）
- **强制转换**：`(int) 3.14`，可能丢失精度
- 运算中 `byte`/`short`/`char` 先提升为 `int`

### 变量与常量

```java
int age = 18;                    // 局部变量须初始化
final double PI = 3.14159;       // 常量，不可重新赋值
static final int MAX = 100;      // 类常量
```

### 作用域

- 局部变量：方法或代码块内
- 成员变量：类内，有默认值
- 块级作用域：`{}` 内有效

### 注释

- 单行 `//`、多行 `/* */`、文档 `/** */`（Javadoc）

## 重点难点

- 基本类型存值，引用类型存地址
- `==` 比较基本类型值，引用类型默认比较地址
