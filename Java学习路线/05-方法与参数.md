# 第 05 章：方法与参数

## 学习目标

- 定义与调用方法
- 理解参数传递机制与方法重载

## 核心知识点

### 方法结构

```java
[修饰符] 返回类型 方法名(参数列表) [throws 异常] {
    // 方法体
    return 结果;
}
```

### 参数传递

- Java **只有值传递**
- 基本类型：传值的副本
- 引用类型：传引用的副本（指向同一对象）

### 方法重载（Overload）

- 同名方法，参数列表不同（类型、个数、顺序）
- 与返回值无关

```java
void print(int x) { /* ... */ }
void print(String s) { /* ... */ }
void print(int x, String s) { /* ... */ }
```

### 可变参数

```java
void sum(int... numbers) {
    for (int n : numbers) { /* ... */ }
}
```

### 递归

- 方法调用自身，需明确终止条件
- 典型：阶乘、斐波那契、树/目录遍历

### 常用修饰符

| 修饰符 | 含义 |
|--------|------|
| `static` | 类方法，不依赖实例 |
| `final` | 不可被子类重写 |
| `abstract` | 抽象方法，无方法体 |

## 实践建议

- 编写工具类 `MathUtil`，练习静态方法与重载
- 用递归实现简单算法，对比迭代写法
