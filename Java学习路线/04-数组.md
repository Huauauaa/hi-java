# 第 04 章：数组

## 学习目标

- 掌握一维、多维数组的声明与使用
- 理解数组内存模型与常见操作

## 核心知识点

### 一维数组

```java
int[] nums = new int[5];           // 默认值 0
int[] arr = {1, 2, 3, 4, 5};       // 字面量初始化
int[] data = new int[]{1, 2, 3};   // 匿名数组
```

### 多维数组

```java
int[][] matrix = new int[3][4];
int[][] grid = {{1, 2}, {3, 4}, {5, 6}};
```

### 数组属性与方法

- `length`：数组长度（非方法，是字段）
- 遍历：`for` 或增强 `for`
- `Arrays.sort(arr)`、`Arrays.toString(arr)`、`Arrays.copyOf(arr, len)`

### 数组与内存

- 数组是引用类型，元素在堆上连续存储
- 赋值传递引用，浅拷贝需注意共享

### 常见算法

- 查找最值、求和、平均值
- 冒泡/选择排序（了解即可，实际用 `Arrays.sort`）
- 二分查找（有序数组，`Arrays.binarySearch`）

## 重点难点

- 数组下标从 0 开始，注意越界 `ArrayIndexOutOfBoundsException`
- 二维数组本质是「数组的数组」，每行长度可不同（锯齿数组）
