# 第 07 章：封装、继承与多态

## 学习目标

- 运用三大特性构建可维护的类层次
- 理解 `super`、方法重写与向上转型

## 核心知识点

### 封装（Encapsulation）

- 隐藏内部细节，通过 public 方法暴露接口
- `private` 字段 + getter/setter
- 好处：可控访问、便于校验与演进

```java
public class Account {
    private double balance;

    public void deposit(double amount) {
        if (amount > 0) balance += amount;
    }

    public double getBalance() { return balance; }
}
```

### 继承（Inheritance）

- `extends` 单继承（Java 不支持多继承类）
- 子类继承父类非 private 成员
- `super`：调用父类构造/方法/字段
- `Object` 是所有类的根

### 方法重写（Override）

- 子类重新定义父类方法
- 规则：同名、同参、返回类型兼容、访问权限不能更严
- `@Override` 注解
- 与重载（Overload）区分

### 多态（Polymorphism）

- **编译看左边，运行看右边**
- 父类引用指向子类对象

```java
Animal a = new Dog();
a.speak();  // 调用 Dog 的 speak()
```

- 多态前提：继承 + 重写 + 向上转型
- 向下转型需 `instanceof` 检查

### `final` 在继承中

- `final class`：不可被继承
- `final method`：不可被重写

## 重点难点

- 构造方法：子类须先调 `super()`（显式或隐式）
- 多态下无法调用子类特有方法，需强转
