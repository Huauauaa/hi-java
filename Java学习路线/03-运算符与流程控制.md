# 第 03 章：运算符与流程控制

## 学习目标

- 熟练使用各类运算符
- 掌握条件、循环与跳转语句

## 核心知识点

### 运算符

| 类别       | 示例                    |
| ---------- | ----------------------- |
| 算术       | `+ - * / % ++ --`       |
| 关系       | `== != > < >= <=`       |
| 逻辑       | `&& \|\| !`（短路）     |
| 位运算     | `& \| ^ ~ << >> >>>`    |
| 赋值       | `= += -= *= /=`         |
| 三元       | `条件 ? 值1 : 值2`      |
| instanceof | `obj instanceof String` |

### 条件语句

```java
if (score >= 60) {
    // ...
} else if (score >= 90) {
    // ...
} else {
    // ...
}

switch (day) {
    case 1 -> System.out.println("Mon");  // Java 14+ 箭头语法
    case 2 -> { /* ... */ }
    default -> { /* ... */ }
}
```

### 循环语句

```java
for (int i = 0; i < 10; i++) { /* ... */ }

for (int n : arr) { /* 增强 for */ }

while (condition) { /* ... */ }

do { /* ... */ } while (condition);
```

### 跳转

- `break`：跳出当前循环或 switch
- `continue`：跳过本次循环
- `return`：结束方法

### 流程控制要点

- `switch` 对 `String`、枚举、部分基本类型有效
- 避免过多嵌套，用 early return 简化逻辑

## 实践建议

- 用循环打印九九乘法表、斐波那契数列
- 练习 `switch` 表达式（Java 14+）替代 if-else 链
