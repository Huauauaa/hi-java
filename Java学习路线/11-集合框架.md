# 第 11 章：集合框架

## 学习目标

- 掌握 List、Set、Map 及其实现类选型
- 理解迭代器、Comparable 与 Comparator

## 核心知识点

### 体系结构

```
Collection
├── List（有序可重复）
│   ├── ArrayList（数组，查询快）
│   └── LinkedList（链表，增删快）
├── Set（无序不重复）
│   ├── HashSet（哈希）
│   ├── LinkedHashSet（插入顺序）
│   └── TreeSet（排序）
└── Queue / Deque
        └── ArrayDeque, PriorityQueue

Map（键值对，键不重复）
├── HashMap
├── LinkedHashMap
├── TreeMap
└── ConcurrentHashMap（并发）
```

### 常用操作

```java
List<String> list = new ArrayList<>();
list.add("a");
list.get(0);
list.remove(0);

Map<String, Integer> map = new HashMap<>();
map.put("key", 1);
map.get("key");
map.getOrDefault("missing", 0);

Set<String> set = new HashSet<>();
set.add("x");
set.contains("x");
```

### 迭代

```java
for (String s : list) { /* ... */ }

Iterator<String> it = list.iterator();
while (it.hasNext()) {
    String s = it.next();
    it.remove();  // 安全删除
}
```

### 排序

- `Collections.sort(list)` / `list.sort(Comparator)`
- `Comparable`：自然排序，实现 `compareTo`
- `Comparator`：外部比较器，`Comparator.comparing(Person::getAge)`

### 泛型集合

```java
List<Integer> nums = new ArrayList<>();
Map<String, List<String>> nested = new HashMap<>();
```

### 选型建议

| 场景 | 推荐 |
|------|------|
| 随机访问多 | `ArrayList` |
| 频繁头尾增删 | `LinkedList` / `ArrayDeque` |
| 去重 | `HashSet` |
| 需排序 | `TreeSet` / `TreeMap` |
| 并发读写 | `ConcurrentHashMap` |

## 重点难点

- `HashMap` 原理：数组 + 链表/红黑树（JDK 8+）
- 重写 `equals` 必须重写 `hashCode`（用于 Set/Map）
- 遍历时不要用增强 for 直接删除（用 Iterator）
