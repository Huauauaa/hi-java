# 第 06 章：类与对象

## 学习目标

- 理解面向对象核心概念
- 掌握类的定义、对象的创建与使用

## 核心知识点

### 类与对象

- **类**：对象的模板（属性 + 行为）
- **对象**：类的实例，通过 `new` 创建

```java
public class Person {
    String name;
    int age;

    void sayHello() {
        System.out.println("Hi, I'm " + name);
    }
}

Person p = new Person();
p.name = "Alice";
p.sayHello();
```

### 构造方法

- 与类同名，无返回类型
- 可重载；未写时编译器提供无参默认构造
- `this()` 调用同类其他构造

### 成员变量 vs 局部变量

| | 成员变量 | 局部变量 |
|---|---------|---------|
| 位置 | 类中 | 方法/块中 |
| 默认值 | 有 | 无，须初始化 |
| 修饰符 | 可用 | 不可用 |

### 访问修饰符

| 修饰符 | 同类 | 同包 | 子类 | 其他 |
|--------|------|------|------|------|
| `private` | ✓ | | | |
| 默认 | ✓ | ✓ | | |
| `protected` | ✓ | ✓ | ✓ | |
| `public` | ✓ | ✓ | ✓ | ✓ |

### 静态成员

- `static` 变量/方法属于类，所有实例共享
- 静态方法不能访问非静态成员（无 `this`）

### 代码块

- 实例块：创建对象时执行
- 静态块：类加载时执行一次

### `this` 关键字

- 指向当前对象
- 区分成员与局部同名变量
- 构造方法链式调用
